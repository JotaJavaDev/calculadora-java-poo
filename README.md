# Calculadora Java POO

Projeto de calculadora desenvolvido em Java utilizando Programação Orientada a Objetos.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)

## Objetivo

Projeto criado para praticar conceitos de:
- Classes
- Objetos
- Métodos
- Encapsulamento

## Como executar

1. Abra o projeto em uma IDE Java
2. Execute a classe `Main.java`
